Symbol 相关

1. 概述

Symbol 是ES6新增的一种数据类型，表示一种“独一无二的值”，任何两个 Symbol 类型的实例都是不相等的。

2. 使用

// 创建实例
let s = Symbol();    // 创建 Symbol 实例
let a = Symbol('a');    // 参数只是实例的描述

// 类型转换
let str = Symbol('str');
String(str);    // 转换为字符串
str.toString();
let flag = Symbol();
Boolean(flag);    // 转换为布尔类型

// 注意事项
// (1) Symbol 实例不能和其他类型的值进行运算

3. 应用

由于 Symbol 的特殊性，在一些场合中很适用。

（1）用作对象的“属性名”；可解决：对象扩展时，属性被覆盖的问题；
